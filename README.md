## üöÄ Overview
An engineering-focused tool designed to optimize arterial traffic signal coordination using **Mixed-Integer Linear Programming (MILP)**. 

### Core Problem
Arterial traffic congestion is often caused by poorly coordinated signal timings. Existing industry tools are either prohibitively expensive or lack intuitive visualization for engineers to perform "what-if" analysis in real-time.

### The Solution
I built a full-stack workstation that:
1.  **Optimizes:** Uses the **Gurobi Optimizer** to maximize "Green Bandwidth" along a corridor using the MAXBAND mathematical model.
2.  **Visualizes:** Renders an interactive **Time-Space Diagram** using a custom SVG engine, making complex data easy to interpret.
3.  **Adjusts:** Allows for manual fine-tuning through drag-and-drop interactions, giving engineers total control.
